v3.0 迁移指南

从 v2.x 平滑迁移到 v3.0，了解重大变更和新特性

v3.0.8 MCP 2025-11-25 SDK 1.27.1

概述

MCP Probe Kit v3.0 是一个重大版本更新，精简优化到 22 个工具，带来了更强大的功能和更好的开发体验。

💡 迁移时间：大多数项目可以在 10-15 分钟内完成迁移，v2.x 的核心工具在 v3.0 中都保持兼容。

主要变更

🛠️ 1. 工具精简优化

v2.x: 39 个工具 → v3.x: 22 个工具（移除低频工具，保留高价值工具）

🔄 2. 新增工作流编排

11 个 start_* 工作流编排工具，提供端到端的自动化开发流程：

  • start_feature - 功能开发工作流
  • start_bugfix - Bug 修复工作流
  • start_ui - UI 开发工作流
  • start_product - 产品设计工作流
  • start_ralph - Ralph Loop 自动化开发

🎨 3. 新增 UI/UX 工具集

完整的 UI/UX 工具链：

  • ui_design_system - 智能设计系统生成器
  • ui_search - UI/UX 数据库搜索
  • sync_ui_data - UI/UX 数据同步
  • init_component_catalog - 组件目录初始化
  • render_ui - UI 渲染引擎

📊 4. 结构化输出升级

核心与编排工具优先支持结构化输出，输出格式一致、可预测，便于工具串联与状态追踪。

迁移步骤

📦 步骤 1：更新版本

Bash 
# 如果使用 npx（推荐）
# 无需操作，自动使用最新版本

# 如果全局安装
npm update -g mcp-probe-kit

📝 步骤 2：检查配置文件

v3.0 的配置文件格式与 v2.x 完全兼容，无需修改。

✅ 向后兼容：v2.x 的核心工具在 v3.0 中都保持兼容，可以无缝升级。
