# MCP Probe Kit v2.0 (vNext) - Product Requirement Document

## 1. 愿景 (Vision)
打造下一代智能开发增强工具集，从 "工具集合" 进化为 "智能开发伙伴"。v2.0 将重点强化 AI 的自主推理、上下文感知和全流程编排能力。

## 2. 核心目标 (Core Objectives)
- **智能化 (Intelligence)**: 引入更强的推理引擎，不仅是 UI，还要覆盖代码重构、Bug 修复等场景。
- **模块化 (Modularity)**: 将工具集拆分为更细粒度的插件系统，支持按需加载。
- **可视化 (Visualization)**: 提供 GUI 界面或更友好的可视化报告（如 HTML 格式的分析报告）。
- **生态化 (Ecosystem)**: 支持第三方插件和社区贡献的 Skills。

## 3. 规划功能 (Planned Features)

### 3.1 智能编排增强
- [ ] **Smart Refactor**: 基于 AST 和 AI 的深度重构，支持跨文件重构。
- [ ] **Auto Debugger**: 自动运行测试 -> 捕获错误 -> 修复 -> 验证的闭环。
- [ ] **Context Aware**: 自动建立项目知识图谱，理解业务逻辑。

### 3.2 UI/UX Pro Max 2.0
- [ ] **Design to Code 2.0**: 支持从截图直接生成代码（多模态）。
- [ ] **Interactive Preview**: 在 CLI 中提供简单的交互式预览或启动本地服务器。
- [ ] **Figma Integration**: 支持从 Figma 导入设计 Token。

### 3.3 开发者体验
- [ ] **Web Dashboard**: 提供一个本地 Web 界面来管理工具和查看报告。
- [ ] **MCP Hub**: 内置插件市场，一键安装社区工具。

## 4. 架构升级 (Architecture Upgrade)
- 迁移至 Monorepo 结构。
- 引入插件化架构。
- 优化性能，减少启动时间。

## 5. 里程碑 (Milestones)
- **Phase 1**: 架构重构 & 插件系统
- **Phase 2**: 核心智能工具升级
- **Phase 3**: Web Dashboard & 可视化
- **Phase 4**: 正式发布 v2.0
